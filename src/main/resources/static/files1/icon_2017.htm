翻转式教学法「学员」作主 讲师与学员角色互换
翻转教室（inverted classroom）的讲师从事翻转教学（flip teaching），而学员则进行翻转学习（flip learning）; 学员先自学线上教材，再於课堂上由讲师解惑、引导学员讨论与实作的模式，翻转了传统，先由讲师在课堂讲授，再由学员回家做作业的形式，故称「翻转式教学法」（Flipped Learning）。
